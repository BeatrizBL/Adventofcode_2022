# Day 14: Regolith Reservoir

Description of the task for this day can be found in https://adventofcode.com/2022/day/14.

## Solution used

The comment of the description that the map "is mostly air" suggests that the matrix to represent the rock lines will be very sparse. Therefore, we use again a dictionary to represent the rock positions, as for day 9.

For the second part, the floor does not need to be infinite as stated in the descriptions. Since the sand can only drop vertically and diagonally, it is enough to add floor tiles for as long as the height of the sand hole (to each side).

Although it is not required, the existing visualization method for sparse matrices was used to produce the same outputs as in the problem description. Here you can see the output for the first part.
